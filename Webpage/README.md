# CompTIA Security+ SY0-701 Study Guide

An interactive, browser-based study guide for the CompTIA Security+ (SY0-701) exam.

**Live site:** https://securityplus-studyguide.vercel.app

## Features

- Chapter-by-chapter study pages (Chapters 1–7)
- Practice quiz with multiple-choice questions
- Performance-Based Questions (PBQ) practice
- Glossary of key terms
- Visual diagrams for each chapter
- AI study assistant (powered by WebLLM, runs locally in the browser)

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Home / chapter select |
| `chapters1-2.html` | Chapters 1–2 |
| `chapter3.html` | Chapter 3 |
| `chapters4-5.html` | Chapters 4–5 |
| `chapter6.html` | Chapter 6 |
| `chapter7.html` | Chapter 7 |
| `practice.html` | Practice quiz |
| `glossary.html` | Glossary |

## Deployment

Deployed on [Vercel](https://vercel.com). Pushes to `main` automatically redeploy the live site.
